## 1. Base Measures

Foundational measures used by other calculations. I standardized all financial metrics on Gross Sales.

### Total Gross Sales

```dax
Total Gross Sales = SUM(fact_transactions[SALES_VALUE])
```

**Purpose:** Primary financial metric representing the final transaction amount.

### Total Transactions

```dax
Total Transactions = COUNTROWS(fact_transactions)
```

**Purpose:** Count of line items.

### Total Baskets

```dax
Total Baskets = DISTINCTCOUNT(fact_transactions[BASKET_ID])
```

**Purpose:** Count of unique shopping trips.

### Average Order Value (AOV)

```dax
Average Order Value = DIVIDE([Total Gross Sales], [Total Baskets], 0)
```

**Purpose:** Average spend per visit.

## 2. KPI Measures (Current State)

These measures utilize my engineered `dim_customer_current` table for high-performance state analysis, avoiding the Cartesian product errors of the legacy snapshot model.

### Active Households

```dax
Active Households = 
CALCULATE(
    COUNTROWS(dim_customer_current),
    dim_customer_current[Current_Status] = "Active"
)
```

**Business Logic:** Households with a purchase in the last 14 days.

### At-Risk Households

```dax
At-Risk Households = 
CALCULATE(
    COUNTROWS(dim_customer_current),
    dim_customer_current[Current_Status] = "At-Risk"
)
```

**Business Logic:** Households in the 15 to 30 day intervention window.

### Churned Households

```dax
Churned Households = 
CALCULATE(
    COUNTROWS(dim_customer_current),
    dim_customer_current[Current_Status] = "Churned"
)
```

**Business Logic:** Households inactive for over 30 days.

### Churn Rate

```dax
Churn Rate = 
DIVIDE(
    [Churned Households],
    COUNTROWS(dim_customer_current),
    0
)
```

**Purpose:** Percentage of the total database that is currently churned.

### At-Risk Gross Sales (LTM)

```dax
At-Risk Gross Sales = 
VAR AtRiskHHs = 
    CALCULATETABLE(
        VALUES(dim_customer_current[household_key]),
        dim_customer_current[Current_Status] = "At-Risk"
    )
RETURN
CALCULATE(
    [Total Gross Sales],
    TREATAS(AtRiskHHs, fact_transactions[household_key]),
    DATESINPERIOD(dim_calendar[Date], MAX(dim_calendar[Date]), -365, DAY)
)
```

**Purpose:** Total Gross Sales over the last 12 months generated by customers who are currently At-Risk. I used TREATAS to pass this virtual filter efficiently.

## 3. Customer Lifecycle Measures

### Days Since Last Purchase

```dax
Days Since Last Purchase = 
VAR AnalysisDate = MAX(dim_calendar[Date])
VAR LastPurchase = MAX(fact_transactions[DATE])
RETURN
DATEDIFF(LastPurchase, AnalysisDate, DAY)
```

**Purpose:** Recency at the filtered context level.

### Lifecycle Stage (Dynamic)

```dax
Lifecycle Stage = 
VAR DaysSince = [Days Since Last Purchase]
RETURN
SWITCH(
    TRUE(),
    DaysSince <= 14, "Active",
    DaysSince <= 30, "At-Risk",
    "Churned"
)
```

**Purpose:** Categorizes current filter context into lifecycle stage.

## 4. RFM and Segmentation Measures

### Avg Recency

```dax
Avg Recency = AVERAGE(dim_rfm[Recency_Days])
```

### Avg Frequency

```dax
Avg Frequency = AVERAGE(dim_rfm[Frequency])
```

### Avg Monetary

```dax
Avg Monetary = AVERAGE(dim_rfm[Monetary])
```

### Segment Gross Sales Contribution

```dax
Segment Gross Sales Contribution = 
DIVIDE(
    [Total Gross Sales],
    CALCULATE([Total Gross Sales], REMOVEFILTERS(dim_rfm[RFM_Segment])),
    0
)
```

**Purpose:** Percentage of total Gross Sales contributed by the selected segment. I used REMOVEFILTERS to clear the visual context.

### RFM Score Display

```dax
RFM Score Display = 
SELECTEDVALUE(dim_rfm[R_Score]) & 
SELECTEDVALUE(dim_rfm[F_Score]) & 
SELECTEDVALUE(dim_rfm[M_Score])
```

**Purpose:** Displays the 3-digit score for tooltips.

## 5. Campaign Measures

### Total Campaigns

```dax
Total Campaigns = DISTINCTCOUNT(dim_campaign[CAMPAIGN])
```

### Coupons Redeemed

```dax
Coupons Redeemed = COUNTROWS(fact_redemptions)
```

### Coupons Sent

```dax
Coupons Sent = 
SUMX(
    fact_campaign_enrollment,
    CALCULATE(COUNTROWS(dim_campaign), dim_campaign[CAMPAIGN] = fact_campaign_enrollment[DESCRIPTION])
)
```

### Redemption Rate

```dax
Redemption Rate = DIVIDE([Coupons Redeemed], [Coupons Sent], 0)
```

### Redeemer Gross Sales Pre

```dax
Redeemer Gross Sales Pre = 
VAR CampaignStart = MIN(dim_campaign[START_DATE])
VAR PreWindow = CampaignStart - 30
VAR Redeemers = VALUES(fact_redemptions[household_key])
RETURN
CALCULATE(
    [Total Gross Sales],
    TREATAS(Redeemers, fact_transactions[household_key]),
    dim_calendar[Date] >= PreWindow,
    dim_calendar[Date] < CampaignStart
)
```

### Redeemer Gross Sales Post

```dax
Redeemer Gross Sales Post = 
VAR CampaignEnd = MAX(dim_campaign[END_DATE])
VAR PostWindow = CampaignEnd + 30
VAR Redeemers = VALUES(fact_redemptions[household_key])
RETURN
CALCULATE(
    [Total Gross Sales],
    TREATAS(Redeemers, fact_transactions[household_key]),
    dim_calendar[Date] > CampaignEnd,
    dim_calendar[Date] <= PostWindow
)
```

### Redeemer Lift %

```dax
Redeemer Lift % = 
VAR Pre = [Redeemer Gross Sales Pre]
VAR Post = [Redeemer Gross Sales Post]
RETURN
DIVIDE(Post - Pre, Pre, 0)
```

## 6. Time Intelligence Measures

### Gross Sales YoY

```dax
Gross Sales YoY = 
VAR CurrentSales = [Total Gross Sales]
VAR PriorSales = CALCULATE([Total Gross Sales], SAMEPERIODLASTYEAR(dim_calendar[Date]))
RETURN
CurrentSales - PriorSales
```

### Gross Sales YoY %

```dax
Gross Sales YoY % = 
VAR Current = [Total Gross Sales]
VAR Prior = CALCULATE([Total Gross Sales], SAMEPERIODLASTYEAR(dim_calendar[Date]))
RETURN
DIVIDE(Current - Prior, Prior, 0)
```

### Gross Sales LTM

```dax
Gross Sales LTM = 
CALCULATE(
    [Total Gross Sales],
    DATESINPERIOD(dim_calendar[Date], MAX(dim_calendar[Date]), -365, DAY)
)
```

## 7. Category Performance Measures

### Category Gross Sales Share

```dax
Category Gross Sales Share = 
DIVIDE(
    [Total Gross Sales],
    CALCULATE([Total Gross Sales], REMOVEFILTERS(dim_product[DEPARTMENT])),
    0
)
```

### Private Label Share

```dax
Private Label Share = 
VAR PrivateSales = CALCULATE([Total Gross Sales], dim_product[BRAND] = "Private")
RETURN
DIVIDE(PrivateSales, [Total Gross Sales], 0)
```

## 8. Promotional Lift Measures

### Baseline Units

```dax
Baseline Units = 
CALCULATE(
    SUM(fact_transactions[QUANTITY]),
    causal_data[DISPLAY] = "0",
    causal_data[MAILER] = "0"
)
```

**Purpose:** Sales volume without any promotional support.

### Promotional Lift %

```dax
Promotional Lift % = 
VAR PromoUnits = SUM(fact_transactions[QUANTITY])
VAR BaseUnits = [Baseline Units]
RETURN
DIVIDE(PromoUnits - BaseUnits, BaseUnits, 0)
```

## 9. Utility Measures

### Selected Date Range

```dax
Selected Date Range = 
FORMAT(MIN(dim_calendar[Date]), "MMM D, YYYY") & " - " & 
FORMAT(MAX(dim_calendar[Date]), "MMM D, YYYY")
```

### No Data Message

```dax
No Data Message = IF(ISBLANK([Total Gross Sales]), "No data for selected filters", BLANK())
```
